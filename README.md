# 图书管理系统 python1216

## 项目概述

图书管理系统是基于Python语言和Django框架，使用sqlite作为数据库开发的系统。本系统适用于各种图书管理场景，提供完整的源代码及配置文档，方便用户理解和使用。

## 技术栈

- **后端**: Python 3.x
- **框架**: Django
- **数据库**: sqlite / Mysql

## 功能模块

### 基础功能

- 登陆功能
  - 用户认证
  - 权限验证

### 核心管理

- 图书管理
  - 图书信息维护
  - 图书分类管理

- 借阅管理
  - 借阅记录
  - 还书提醒

- 个人管理
  - 个人资料维护
  - 借阅历史查询

- 会员管理
  - 会员信息维护
  - 会员借阅权限设置

### 数据分析

- 数据可视化图表
  - 图书借阅统计
  - 会员活跃度分析

## 系统角色

- 系统管理员
- 普通用户
- 会员用户

## 运行环境

- Python 3.x
- Django 框架
- sqlite 或 Mysql 数据库
